# RedPacket
-    红包助手，QQ、微信、支付宝自动抢红包源码

## 一、功能介绍：（功能正在完善中，若有不完善的地方，请邮箱联系）
>   a、开启QQ红包服务，可自动抢普通红包，口令红包。自动拆开，抢过的红包自动过滤;
>   b、开启WeChat红包服务，可自动抢普通红包，口令红包。自动拆开，抢过的红包自动过滤;
>   c、开启AliPay红包服务，可自动抢咻红包，但是不能自动打开;

## 三、使用介绍：
-   ①app使用只需打开QQ、WeChat、AliPay服务即可
-   ②代码使用请参考代码注释
